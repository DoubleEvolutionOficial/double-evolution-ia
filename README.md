# Double Evolution IA

## Versão
0.1.0-alpha

## Objetivo

Plataforma de análise estatística baseada em regras próprias para estudo de históricos da Double.

## Estrutura do Projeto

- `backend/` - serviço FastAPI, configuração de banco e módulos iniciais.
- `frontend/` - aplicação React com Vite.
- `docs/` - documentação arquitetural.
- `tests/` - testes automatizados.
- `docker/` - compose para desenvolvimento com backend e frontend.

## Tecnologias

- Python
- FastAPI
- React
- PostgreSQL
- Docker
