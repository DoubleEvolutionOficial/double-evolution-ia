# Sprint 6 - Módulo Laboratory

## Objetivo

Criar o módulo Laboratory como uma camada independente para registrar eventos de análise e preparar a base para futuras análises estatísticas.

- Implementar registro simples de eventos.
- Expor estatísticas básicas de eventos.
- Manter a arquitetura sem acoplamento ao RuleEngine.
- Não implementar descoberta de padrões ainda.
