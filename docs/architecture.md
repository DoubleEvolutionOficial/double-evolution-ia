# Arquitetura do Projeto

Este documento descreve a estrutura planejada para o projeto Double Evolution IA.

- `backend/` - serviço FastAPI com arquitetura modular e separação de responsabilidade.
- `frontend/` - aplicação React + Vite para consumir a API.
- `docs/` - documentação arquitetural e guias iniciais.
- `tests/` - testes de integração e unidade.
- `docker/` - configuração de orquestração e exemplos de implantação.
